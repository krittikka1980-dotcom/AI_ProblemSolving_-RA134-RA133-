# Tic Tac Toe AI 🎮

## Description
This project implements an interactive Tic Tac Toe game with an AI opponent using Minimax Algorithm.

## Features
- Player vs AI gameplay
- Wooden themed UI
- Animated moves
- Win / Draw detection

## Algorithms Used
- Minimax Algorithm

## How to Run
Open index.html in browser.

## Output
User can play against an unbeatable AI system.

# 🎮 Tic Tac Toe AI (Minimax vs Alpha-Beta)

## 📌 Project Description
This project implements an interactive Tic Tac Toe game where a human player competes against an AI opponent. The AI always plays optimally using advanced Artificial Intelligence algorithms.

The system also compares the performance of two algorithms:
- Minimax Algorithm  
- Alpha-Beta Pruning  

---

## ✨ Features
- 🎨 Wooden themed UI with animations  
- 🌿 Animated background (leaves effect)  
- 🤖 Player vs Unbeatable AI  
- 🔄 Algorithm selection (Minimax / Alpha-Beta)  
- 📊 Performance comparison:
  - Nodes explored  
  - Execution time  
- 🏁 Win / Draw detection  
- 🔁 Restart functionality  

---

## 🧠 Algorithms Used

### 1. Minimax Algorithm
- Explores all possible game states  
- Guarantees optimal move  
- Higher computation cost  

### 2. Alpha-Beta Pruning
- Optimized version of Minimax  
- Prunes unnecessary branches  
- Faster and more efficient  

---

## ⚖️ Comparison

| Feature            | Minimax        | Alpha-Beta       |
|--------------------|---------------|------------------|
| Nodes Explored     | High          | Reduced          |
| Execution Time     | Slower        | Faster           |
| Efficiency         | Moderate      | High             |
